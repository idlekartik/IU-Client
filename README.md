# IU Client 1.0.0

Professional Minecraft Client for Minecraft Java 1.21.11 Fabric.

## Features
- FPS Booster
- PvP Optimizer
- HUD Editor
- Armor and Potion HUD
- Music Player
- Favorite Servers
- Better Server Tools
- Post-Fight Coach
- IU Pulse Animations

## Installation
1. Download IU Client from Releases.
2. Extract the downloaded ZIP.
3. Run `Install IU Client.bat`.
4. Open Minecraft Launcher.
5. Select Fabric 1.21.11 and Play.
