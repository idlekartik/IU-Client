Install in minutes

Download. Install.
Launch IU Client.

The public package places IU Client, configs and required assets into the correct Minecraft folders automatically.

Get the latest release
  1. 01
    Download the release

    Open GitHub Releases and download the IU Client public ZIP.

  2. 02
    Extract completely

    Right-click the ZIP and extract it before running the installer.

  3. 03
    Run the installer

    Open Install IU Client.bat and let it place all files correctly.

  4. 04
    Launch Fabric 1.21.11

    Open Minecraft Launcher, select Fabric 1.21.11 and press Play.

Quick answers

Frequently asked
questions.

Which Minecraft version does IU Client support?

The current public release is built for Minecraft Java 1.21.11 with Fabric Loader.

Is IU Client free?

Yes. Download the current release from the official GitHub repository.

Where do I place my songs?

Use the Music Player's Open Folder button, or place supported local audio files inside .minecraft/config/iuclient/music/.

Can I move and resize HUD elements?

Yes. Open the Full HUD Editor to move and scale each supported widget independently.
